# Ayesha — Developer Portfolio

A bold, editorial-style portfolio website for **Ayesha**, a WordPress Developer & Frontend Specialist based in Lahore, Pakistan. Built around oversized typography, scroll-driven motion, and a signature set of interactive mechanics rather than a conventional "developer portfolio" template.

**Live design direction:** white / off-white base with green and blue accents (light theme), full dark-mode parity via a theme toggle.

---

## ✨ Features

- **Cinematic loading sequence** *(later removed per design decision — see Changelog)*
- **Magnetic, marquee-style full-screen navigation menu** triggered by a pill button
- **Custom GSAP-driven hero** with masked text reveal and mouse-parallax tilt
- **Scroll-pinned "Manifesto" statement** with blur-to-sharp scrub reveal
- **Selected Work** — a pinned, scroll-scrubbed depth-peel card stack on desktop; a swipeable single-card view with arrow/dot navigation on mobile
- **Mini Services** list with a hover-revealed live preview panel (recreated WordPress admin, code, and WooCommerce/Elementor UI snippets)
- **Interactive Tech Stack constellation** — a force-directed, mouse-reactive node web with WordPress as the central hub
- **Reveal-style Contact/Footer** section, pinned behind the page and uncovered as the user scrolls past About
- **Light/dark theme toggle** with a fully tokenized color system (zero hardcoded colors) and an animated transition flash
- **Custom scroll-progress indicator** (left edge of viewport) replacing the default browser scrollbar
- **Full SEO layer** — metadata, Open Graph/Twitter cards, JSON-LD Person/ProfilePage schema, semantic headings, sitemap, and robots.txt
- **Fully responsive**, with mobile-specific behavior for every scroll-jacked/pinned desktop effect

| Layer | Technology |

---

## 🎨 Design Language

- Oversized, condensed typography paired with editorial mixed-weight headlines
- Minimal, confident negative space over dense grids or card-heavy layouts
- One signature interactive mechanic per major section (constellation, depth-peel, marquee menu, preview panels) rather than repeating a single effect site-wide
- Light theme as default; dark theme as a fully-supported, equally polished alternative — not an afterthought

---

## 📱 Responsiveness

All cursor-dependent and scroll-jacking effects (depth-peel stack, magnetic constellation, parallax tilt) are disabled or replaced with touch-appropriate equivalents below 1024px — including a dedicated swipe/arrow-based project navigator and tap-to-expand project detail views on mobile.
